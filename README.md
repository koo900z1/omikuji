# Omikuji
pythonで作成したくじアプリ。
このプログラムは、運勢を占うくじ引きゲームです。以下の手順で遊ぶことができます。
1. プログラムを実行すると、ウィンドウが表示されます。
2. 「単発」ボタンをクリックすると、今日の運勢が表示されます。
3. 「10連」ボタンをクリックすると、10回分の運勢が順番に表示されます。表示中はボタンが隠れます。
4. 「終了」ボタンをクリックすると、プログラムが終了します。
5. 「使い方」ボタンを押すと別ウィンドウで使い方が出てきます。ただし文言はここと一字一句ほとんど同じです。
運勢の種類と確率
- 大吉: 1%
- 吉: 9%    
- 中吉: 15%
- 小吉: 25%
- 末吉: 25%
- 凶: 15%
- 大凶: 10%

関数
- fortune(): 運勢をランダムに判定して返す関数。
- single_draw(): 単発くじを引いて結果を表示する関数。
- ten_draw(): 10連くじを引いて結果を順番に表示する関数。
- end_program(): プログラムを終了する関数。
- hide_buttons(): ボタンを隠す関数。
- show_buttons(): ボタンを再表示する関数。

注意
このプログラムは、PythonのTkinterライブラリを使用してGUIを作成しています。Pythonがインストールされている環境で実行してください。
運勢の結果は完全にランダムであり、実際の運勢を反映するものではありません。楽しんで遊んでください！

創作主より
GUI化にはAIを使いましたが、運勢の判定や確率設定は自分で考えました。単発と10連の両方を実装することで、より楽しめるように工夫しました。今後も面白いプログラムを作っていきたいと思います。
大吉が出るまで終われない、とか遊んでみると意外と沼ります。時間を溶かす可能性があるので、ほどほどに楽しんでくださいね。
